URW DIN FONT FILES — PLACE THEM HERE
=====================================

Drop your licensed URW DIN font files into this folder.
The app is already wired to load them automatically.

REQUIRED FILE NAMES (rename your files to match exactly):
----------------------------------------------------------
urw-din-regular.woff2       — Regular (weight 400)
urw-din-regular.woff        — Regular (weight 400) fallback
urw-din-medium.woff2        — Medium  (weight 500)
urw-din-medium.woff         — Medium  (weight 500) fallback
urw-din-bold.woff2          — Bold    (weight 700)
urw-din-bold.woff           — Bold    (weight 700) fallback
urw-din-italic.woff2        — Italic  (weight 400, italic)
urw-din-italic.woff         — Italic  (weight 400, italic) fallback

WHERE TO GET URW DIN:
---------------------
- Adobe Fonts (subscription): https://fonts.adobe.com/fonts/urw-din
- MyFonts (purchase):         https://www.myfonts.com/search/urw+din
- URW Type Foundry:           https://www.urwtype.com/

ONCE FILES ARE PLACED:
----------------------
Restart the dev server (npm run dev) and URW DIN will load
automatically across the entire application.
